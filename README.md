## 🏠 HBnB – Holberton AirBnB Clone Project

HBnB is a full-stack web application inspired by AirBnB, developed as part of the Holberton School curriculum.
The project is divided into multiple parts, each focusing on a different layer of backend architecture and application design.

Throughout this project, we implement:

Object-Oriented Programming principles

RESTful API development with Flask

Layered architecture (API, Business Logic, Persistence)

Design patterns such as the Facade pattern

Modular and scalable project structure

Each part of the project builds upon the previous one, progressively transforming a simple architecture into a structured and extensible web application.
